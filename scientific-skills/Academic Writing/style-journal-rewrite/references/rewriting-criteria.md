# Rewriting Criteria

## Rewriting Objective

Transform the draft into the target writing style or target journal's explicit format, without turning the task into expansion, paraphrasing, filler polishing, or content restructuring.

## Hard Constraints

- Do not add facts, data, examples, or citations not present in the original draft
- Do not change the original draft's conclusions, positions, commitments, or temporal relationships
- Do not directly transplant specific expressions from the example text into the result
- Do not sacrifice readability or information accuracy for the sake of "sounding like" the target
- Do not present format details unsupported by the user's materials as mandatory journal requirements

## Specific Operations

### When the target style is more formal

- Increase sentence completeness, reduce colloquial connectors
- Tighten subjective exclamations and colloquial repetitions
- Replace casual expressions with more stable written forms

### When the target style is more colloquial

- Reduce jargon stacking and clause depth
- Allow more natural pauses and transitions
- Preserve clarity without becoming deliberately loose or rambling

### When the target style has more editorial/commentary feel

- Allow leading with conclusions
- Allow moderate evaluative language, but do not artificially strengthen positions
- Build argument strength through contrast, progression, and turns

### When the target style is more restrained

- Remove exaggerated descriptions and vague elevations
- Tone down exclamations, slogans, and declarative statements
- Let judgments emerge from the organization of facts, not from emotionally charged words

### When the target is journal format adaptation

- Prioritize abstract structure, keyword count, body section order, heading hierarchy, and reference style
- When submission guidelines and sample articles conflict, submission guidelines take precedence
- Formats consistently repeated across multiple sample articles serve as strong references
- Occasional patterns in a single sample article serve only as weak references
- If the original draft lacks information needed for a section, keep a placeholder or brief note — do not fabricate content

### When adjusting the abstract

- First determine whether the journal requires a single-paragraph abstract or a structured abstract
- When word limits apply, first preserve research objectives, methods, core results, and conclusions, then compress secondary background
- If sectioned or sub-headed format is required, organize by the journal's specified section names — do not invent new sections

### When adjusting references

- Only convert citation style within the range supported by existing information
- Preserve original fields such as authors, title, journal name, year, volume, issue, and pages
- When original information is missing, do not fabricate abbreviations, page numbers, DOIs, or publication locations
- Simultaneously adjust the correspondence between in-text citations and the reference list at the end
- **Key distinction: superscript is character conversion, not text replacement**. If the target format uses superscripts, first check which superscript implementation the reference article uses (Unicode characters vs. font formatting), then check how consecutive citations are combined (range merging / comma separation / concatenation), then replicate exactly

## Common Failures

- Only replacing a few synonyms without changing the overall character
- Over-imitating the example, resulting in content borrowing
- Adding information without authorization for the sake of fluency
- Language appears sophisticated but is emptier than the original
- Claiming journal format adaptation but only changing tone, without addressing abstract, structure, or references
- Partially converting reference style, leaving in-text citations and the reference list inconsistent

## Pre-delivery Review

Check two questions for each paragraph:

1. Has any information in this paragraph been lost or altered?
2. Does this paragraph truly match the target style, rather than being generic polishing?
3. If there are journal requirements, does this section show clearly visible format adaptation?
