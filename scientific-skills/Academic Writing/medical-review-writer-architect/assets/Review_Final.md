# Sample Medical Review Output

This file previously contained a Chinese-language sample medical review article about the CXCR4/CXCL12 axis in intestinal inflammatory diseases.

The sample demonstrated the expected output format of this skill: a structured review article with PMID citations, organized into sections (Abstract, Introduction, Body Sections 1-6, Conclusion).

## Expected Output Structure

1. Title
2. Abstract + Keywords
3. Introduction (with PMID citations)
4. Body Sections (numbered, with subsections)
5. Each section contains cited claims in format: statement[PMID:XXXXXXXX]
6. Conclusion

## Citation Format

In-text: `[PMID:12345678]`
Multiple: `[PMID:12345678][PMID:23456789]`
