# Medical Review Skill

This is a workflow engine specifically designed for writing long-form, high-quality medical reviews.

## Features

- Step-by-step architecture: Outline construction -> PubMed retrieval -> Chapter-by-chapter drafting -> Conclusion summary -> Abstract finalization -> Citation formatting
- Automated PubMed search and citation management
- Strict citation standards and format checking

For detailed instructions, see `SKILL.md`.
